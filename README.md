# Forecasting-Financial-Inclusion-in-Ethiopia
Build a forecasting system that tracks Ethiopia's digital financial transformation using time series methods

## Running the Dashboard
1. Activate venv: `source .venv/Scripts/activate` (Git Bash)
2. Install deps: `pip install streamlit pandas plotly seaborn matplotlib`
3. Run: `streamlit run dashboard/app.py`
Access at http://localhost:8501
